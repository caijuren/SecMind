## 二、软件用途与技术特点

### 主要功能

SecMind 是一款面向企业安全运营团队的 AI 自主安全研判平台，提供以下核心能力：

1. **信号感知与告警管理**：多源安全信号接入、告警聚合、风险分级
2. **AI 自主研判**：基于大语言模型的攻击链推理、威胁判定、处置建议生成
3. **案件管理与应急响应**：安全事件全生命周期管理、自动化剧本编排
4. **态势感知与报表**：实时安全态势大屏、定时报表生成与导出
5. **知识库与学习系统**：威胁情报知识库、AI 反馈学习与模型优化
6. **多租户管理**：企业级多租户隔离、RBAC 权限控制、合规审计

### 技术特点

- 基于 Next.js 16 + FastAPI 的前后端分离架构
- 深色主题设计系统（Dark-First），支持主题切换
- WebSocket 实时数据推送
- JWT + Refresh Token 轮换认证机制
- 多租户数据隔离（PostgreSQL RLS + ContextVar）
- AI 模型路由与多模型并发调用
- 国际化和本地化支持

## 五、申请流程

1. 登录中国版权保护中心官网（www.ccopyright.com.cn）
2. 注册账号并实名认证
3. 在线填写申请表
4. 上传鉴别材料
5. 提交审核（受理通知书约 3-5 个工作日）
6. 审查（约 30 个工作日）
7. 登记公告与证书发放
