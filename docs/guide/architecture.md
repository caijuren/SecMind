## 数据流

### 告警处理流程

```
外部安全设备 → 数据接入层 → 告警标准化 → 告警存储
                                              │
                                              ▼
                                    AI 自动研判 ← 情报关联
                                              │
                                              ▼
                                    告警分级分类
                                     │       │
                                     ▼       ▼
                              自动响应执行   人工分析
                                     │       │
                                     ▼       ▼
                                    事件闭环 & 报告生成
```

### AI 分析流程

```
用户请求 / 自动触发
        │
        ▼
   任务路由引擎 ──→ 选择最优模型
        │
        ▼
   上下文构建 ──→ 告警数据 + 情报 + 历史分析
        │
        ▼
   LLM 推理 ──→ 结构化输出
        │
        ▼
   结果验证 ──→ 置信度评估
        │
        ▼
   输出呈现 ──→ 分析报告 / 响应建议
```

### 剧本执行流程

```
触发条件满足
      │
      ▼
  DAG 解析 ──→ 构建执行图
      │
      ▼
  节点调度 ──→ 拓扑排序执行
      │
      ├──→ 并行节点：并发执行
      │
      ├──→ 条件节点：分支判断
      │
      ├──→ 人工节点：等待审批
      │
      ▼
  结果聚合 ──→ 执行报告
```

## 核心模块

### AI 分析引擎

AI 分析引擎是 SecMind 的核心模块，负责：

- **告警研判**：自动分析告警，给出威胁评估和建议
- **威胁分析**：深度分析攻击手法、影响范围和关联线索
- **智能建议**：基于上下文生成响应处置建议
- **报告生成**：自动生成结构化安全分析报告

### 剧本编排引擎

基于 DAG（有向无环图）的剧本编排引擎：

- 支持可视化拖拽编排
- 支持条件分支和循环
- 支持并行执行和子剧本调用
- 支持人工审批节点
- 支持超时和异常处理

### 情报关联引擎

负责威胁情报的采集、关联和分发：

- 支持多种情报源接入
- IOC 自动匹配和关联
- 情报缓存与更新策略
- STIX/TAXII 标准支持
